Docker 容器自定义 hosts 网络访问

Docker自定义hosts网络访问

需求：使用自己的域名服务。

在很多应用环境中都会有其他服务器的访问需求。直接使用ip不便于修改。因此搭建自己的域名服务，之后配置docker环境使用此域名服务即可。也可同时使用其它域名服务。

  • 搭建Docker私有仓库

  • 从另外的docker环境获取上述镜像

Docker的镜像归纳为两种

  • 需要知道docker容器的地址的
    类似zk这种组件的
  • 不需要知道docker容器地址
    类似应用程序
    通过注册服务到zk，然后自动发现服务。
